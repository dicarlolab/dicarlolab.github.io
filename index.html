    DLData Public

    In order to stimulate the development of encoding models of visual cortex we have made several large scale neural datasets publicly available via a github repository. This site is intended to describe the format of those datasets, how to submit the predictions of encoding models, and how results will be calculated and posted.

    Data format

    We have modeled the format of the datasets after skdata, a format for standard machine learning datasets.

    Each dataset is a python object which has attached

    • images (as an skdata lazy array)
    • metadata about these images (as a tabular array)
    • neural data (as a numpy array), a vector of responses for each image
    import dldatapublic.neural_datasets as nd
dataset = nd.PNAS2010()
images = dataset.images  #not yet loaded in memory
im0 = images[0]  #loaded in memory as a numpy array
meta = dataset.meta
print meta['category'][0]  #'car'
print meta['obj'][0]  #'ferrari'
IT_neural_features = dataset.IT_neural_features
V4_neural_features = dataset.V4_neural_features

    For further documentation, see the tutorials folder. There is a short readme describing the nature of each dataset. Depending on the dataset, neural data is available at a finer grain than simply a vector of responses per image.

    Submitting encoding models

    • Clone and install the dldata_public repository
    • Generate and save an array for one of the test datasets, the same shape as dataset.neural_features
    • Submit a pull request

    We require that the code used to generate predictions be made publicly available (on github or otherwise) so that others may run your model on similarly structured datasets. Once we verify these conditions, we will evaluate the models for their predictivity and post the results.

    Results/Methodology

    Your predictions will be evaluated using a single-site % variance explained metric, as well as a population level representation dissimilarity metric.
