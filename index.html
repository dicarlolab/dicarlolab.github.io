DLData Public

In order to stimulate the development of encoding models of visual cortex we have made several large scale neural datasets publicly available. This site is intended to describe the format of those datasets, and the conditions that encoding models must meet in order to be evaluated.

Data format

We have modeled the format of the datasets after skdata, a format for standard machine learning datasets.

Each dataset is a python object which has attached

  • images (as an skdata lazy array)
  • metadata about these images (as a tabular array)
  • neural data (as a numpy array), a vector of responses for each image
import dldatapublic.neural_datasets as nd
dataset = nd.PNAS2010()
images = dataset.images  #not yet loaded in memory
im0 = images[0]  #loaded in memory as a numpy array
meta = dataset.meta
print meta['category'][0]  #'car'
print meta['obj'][0]  #'ferrari'
neural_features = dataset.neural_features

For further documentation, see the tutorials folder. There is a short readme describing the nature of each dataset. Depending on the dataset, neural data is available at a finer grain than simply a vector of responses per image.

Submitting encoding models

  • Clone and install the dldata_public repository
  • Generate and save an array for one of the test datasets
  • Submit a pull request

We require that encoding models be available publicly (on github or otherwise), and that the code used to generate features be made public within that repository. Once we verify these conditions, we will evaluate the models for their predictivity and add their features to the python objects.

Publications

You are free to explore the data as you wish, but we ask that you contact us before publishing any analysis of this data.
